História da web

A web é um serviço criado por TIm Bernes Lee, 1990. Que funciona sobre a internet, que é uma rede de computadoresm utilizando a arquitetura Cliente

Para isso ele definiu a estrutura de funcionamento web, cliente servidor, e criou a linguagem HTML, para exibição de documentos e o protocoloi HTTP para comunicação entre a aplicação cliente e servidor.

Para cuidar dos padrões da web e propriedades da tecnologia ele criou a fundação w3c World Web Consorcium

A referência das tags HTML pode ser encontrada no site da w3c
